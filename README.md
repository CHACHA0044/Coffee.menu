## ğŸš€ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/chacha0044/Coffee.menu.git
   cd Coffee.menu
   ```

2. **Add your images**
   - Place your background image in `images/background.jpg`
   - Add coffee images in `images/coffee/` folder
   - Add dessert images in `images/desserts/` folder

3. **Open in browser**
   - Simply open `index.html` in any modern web browser
   - No server setup required!

## ğŸ¨ Customization

### Change Colors
Edit the CSS variables in `menu-styles.css`:
- Primary brown: `#8b4513`
- Background gradient: `rgba(222, 184, 135, 0.97)`
- Text colors: `#4a2511`, `#5a3621`

### Update Menu Items
Modify the HTML structure in `index.html` to add/remove items or change prices.

### Adjust Animations
Change animation duration and timing in the `@keyframes` sections of the CSS file.
